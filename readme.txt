== Description ==
This WordPress plugin facilitates lead integration with the Ágin CRM platform. It is designed for use by Ágin clients who wish to seamlessly send leads generated on their WordPress websites directly to the Ágin CRM system. The plugin sends leads to our API at https://api.agin.com.br/api/AdicionarLead.
